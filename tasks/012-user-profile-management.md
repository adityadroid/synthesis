# 012: User Profile Management

## Phase
Phase 1 (v1.0.0)

## Description
Allow users to view and edit their profile settings.

## Requirements
- GET /users/me endpoint
- PATCH /users/me for updates
- Editable fields: display name, avatar URL
- Return current profile data

## Acceptance Criteria
- [ ] Can view own profile
- [ ] Can update display name
- [ ] Can update avatar URL
- [ ] Changes persist to database

## Dependencies
- 005: Protected Routes Middleware

# 032: Copy Message Button

## Phase
Phase 1 (v1.0.0)

## Description
One-click button to copy entire message.

## Requirements
- Copy button on message hover
- Copy raw text to clipboard
- Visual feedback on copy
- Tooltip explaining action

## Acceptance Criteria
- [ ] Button appears on hover
- [ ] Copies full message text
- [ ] Shows "Copied!" feedback
- [ ] Works on mobile tap

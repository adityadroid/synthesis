# 053: Adjustable Font Size

## Phase
Phase 2 (v1.1.0)

## Description
Allow users to adjust message font size.

## Requirements
- Font size slider/selector
- Minimum/maximum bounds
- Affects message content
- Persists per user

## Acceptance Criteria
- [ ] Can adjust font size
- [ ] Preview updates live
- [ ] Consistent across app
- [ ] Accessible minimum size

## Dependencies
- 051: Light/Dark Theme

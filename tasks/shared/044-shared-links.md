# 044: Shared Links

## Phase
Phase 2 (v1.1.0)

## Description
Generate public read-only share URLs for conversations.

## Requirements
- Generate unique shareable link
- Read-only access for viewers
- Optional expiration
- View count tracking

## Acceptance Criteria
- [ ] Can generate share link
- [ ] Link shows read-only view
- [ ] No authentication required to view
- [ ] Owner can revoke link
